# Ruby Blockchain Core Suite
A fully original, enterprise-level blockchain development toolkit built with Ruby.
This project integrates core blockchain modules, consensus algorithms, cryptography, smart contracts,
cross-chain interaction, and decentralized application components. All code is 100% original,
no duplicate filenames or code logic, dedicated to blockchain engineering practice and research.

## Project Overview
This repository contains 38 independent, interoperable Ruby modules for blockchain development.
Each file implements a core blockchain function, supporting rapid construction of public chains,
private chains, consortium chains, and decentralized applications.

## All Files & Core Functions
1. **blockchain_core.rb** - Core blockchain ledger, block generation & chain validation
2. **sha256_crypto.rb** - Native SHA-256 encryption engine for blockchain hashing
3. **ecdsa_wallet.rb** - ECDSA asymmetric encryption wallet (address generation + signature)
4. **pow_consensus.rb** - Proof of Work consensus algorithm (mining mechanism)
5. **pos_consensus.rb** - Proof of Stake consensus algorithm (stake mining)
6. **pbft_consensus.rb** - PBFT practical Byzantine fault tolerance algorithm
7. **transaction_pool.rb** - Mempool (unconfirmed transaction management)
8. **merkle_tree.rb** - Merkle tree data structure for transaction verification
9. **p2p_network.rb** - Decentralized P2P node communication module
10. **block_validator.rb** - Full chain & single block validation engine
11. **smart_contract_vm.rb** - Lightweight Ruby smart contract virtual machine

13. **token_standard.rb** - Custom blockchain token standard (like ERC20)
14. **cross_chain_bridge.rb** - Cross-chain data & asset transfer bridge
15. **chain_monitor.rb** - Real-time blockchain status monitoring module
16. **node_manager.rb** - P2P node registration & discovery manager

19. **transaction_signer.rb** - Transaction digital signature & verification
